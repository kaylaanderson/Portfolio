Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

puts "Enter degrees in Celsius:"
celsius = gets.to_f 
def celsius_to_fahrenheit(celsius)
    fahrenheit = celsius * 1.8 + 32
end

puts "The temperature is #{celsius_to_fahrenheit(celsius)} degrees Fahrenheit"

def ordinal(right_digit)

  if right_digit == 1
    return "st" 
  elsif right_digit == 2
    return "nd"
  elsif right_digit == 3
    return "rd"
  else
    return "th"
  end
end

puts "Enter a number."
number = gets.chomp.to_i
right_digit = number % 10 || number % 100

puts "That's the #{number}#{ordinal(right_digit)} example."

Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

puts "How many items do you want to see?"
number = gets.chomp.to_i 

number.times do |number|
    number +=1
    number_divisible_by_3 = number % 3
    number_divisible_by_5 = number % 5
  
  
  if number_divisible_by_3 == 0 && number_divisible_by_5 == 0
    puts "Foobar"
  elsif number_divisible_by_3 == 0
    puts "Foo"
  elsif number_divisible_by_5 == 0
    puts "Bar"
  else
    puts number 
  end
end
